SAuto-schedule all tasks

                            
?This help

                            
EscClose modal / cancel inline add / leave input

                        
                    

                    
Tasks (board, list & overview)

                    

                        
                            
Click a taskEdit it

                            
Ctrl+Click a taskMark done / not done

                            
Shift+Click a taskFreeze / unfreeze (scheduler won't move it)

                            
Drag a cardMove between board columns (updates status)

                            
Alt+Click a card (board)Add/remove task to multi-selection (across all columns)

                            
Drag a selected cardMove every selected task to the target column

                            
Enter (board, with selection)Force all selected tasks to done

                            
Ctrl+C (board, with selection)Copy selected tasks as - **Title**: description markdown

                            
Esc / empty click / view switchClear the multi-selection (also clears after each action)

                            
Enter in a column's + inputCreate task in that column

Ctrl+Click an eventMark done / not done

                            
Shift+Click an eventFreeze / unfreeze

                            
Ctrl+Click a day headerFreeze / unfreeze the whole day
